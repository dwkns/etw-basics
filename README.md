# A very basic starter for Eleventy, TailwindCSS v4

## Installation 
Install dependecies
```bash
$ npm install
```
## Development
Start the Development Server
```bash
$ npm run dev
```

## Build 
Build the site

```bash
$ npm run build
```
Serve the static build (useful for local testing)
```bash
npx serve dist
```
