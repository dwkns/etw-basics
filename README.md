# A very basic starter for Eleventy, TailwindCSS v4

## Installation 
Install dependecies
```bash
$ yarn install 
```
## Development
Start the Development Server
```bash
$ yarn dev
```

## Build 
Build the site

```bash
$ yarn build
```
Serve the static build (useful for local testing)
```bash
npx serve dist
```
